Základy JS - opakování

1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

JavaScript je skriptovací jazyk, používá se nejčastěji na webových prohlížečích.

Javascript není jazyk Java, ale používá podobnou syntax příkazů. Oficiálně se nazývá ECMAScript.

Využití:

  • Používá se k vytváření interaktivních stránek (uživatel může ovlivnit jejich podobu).
  • Umožňuje vytvořit webové aplikace na straně klienta (browser) i serveru (NodeJS).
  • Umožňuje tvořit dynamické stránky (změna vlastností, animace).
  • Nabízí mnoho knihoven k nejrůznějšímu použití.
  • Lze vytvářet grafické aplikace včetně her.

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Ending statements with semicolon is not required, but highly recommended.
  2. JavaScript keywords are reserved words. Reserved words cannot be used as names for variables.
  3. Single line comments start with //. Multi-line comments start and end with /* */.
  4. JavaScript ignores multiple spaces. You can add white space to your script to make it more readable.
  5. For best readability, programmers often like to avoid code lines longer than 80 characters.
  6. JavaScript statements can be grouped together in code blocks, inside curly brackets. The purpose of code blocks is to define statements to be executed together.

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

=, +=, -=, *=, /=, %= (Operátor přiřazení přiřadí proměnné zbytek.), **=

Logické

&& (a), || (nebo), ! (ne)

Porovnávací

==, ===, !=, !==, >, <, >=, <=, ? (ternární operátor)

Bitové

&, |, ~, ^, <<, >>, >>> (nepodepsaný pravý posun)

Typové

typeof (Vrátí typ proměnné.), instanceof (Vrátí hodnotu true, pokud je objekt instancí typu objektu.)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* vypíše upozornění do „vyskakovacího“ (popup) okna*/
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
var heslo = prompt('Zadej heslo');

confirm()

/* Vyskočí okno jestli chceš přejít na stránku Seznam.cz po potvrzení přejde. */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování:
Když se objeví výstražné okno, uživatel bude muset pokračovat kliknutím na tlačítko "OK".
Výstražná okna se nepoužívají na bezpečných stránkách a hlavně se automaticky blokují.
